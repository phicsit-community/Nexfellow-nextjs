================================================================================
                         ⚠️ TRICKY PARTS
================================================================================

🔴 HIGH RISK:
   1. Socket.io - Must persist across routes, client-only
   2. Auth Hydration - localStorage not available in SSR
   3. Conditional Routes - Same URL, different component by auth
   4. Theme FOUC - Flash of wrong theme on load

🟡 MEDIUM:
   5. react-helmet → Next.js Metadata API
   6. import.meta.env → process.env.NEXT_PUBLIC_*
   7. React.lazy() → Next.js dynamic()
   8. Docs Layout - Nested routing with sidebar

🟢 EASY:
   9. Link (to → href)
   10. useNavigate → useRouter
   11. useLocation → usePathname
